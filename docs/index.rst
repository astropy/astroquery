###################################################################
Astroquery: Tools for querying astronomical web forms and databases
###################################################################

.. |clearfloat|  raw:: html

    <div class="clearer"></div>

|clearfloat|

This is the documentation for the Astroquery coordinated package of `astropy
<http://www.astropy.org>`__.

Code and issue tracker are on `GitHub <https://github.com/astropy/astroquery>`_.
    
.. Important:: If you use astroquery for work presented in a publication or talk, please
    `cite <https://github.com/astropy/astroquery/blob/main/astroquery/CITATION>`__
    the paper
    `Ginsburg, Sip≈ëcz, Brasseur et al 2019 <https://ui.adsabs.harvard.edu/abs/2019AJ....157...98G/abstract>`_.
